# Botkit

ボットを作成するためのフレームワーク。  
node.jsで動作する。

## インストール

```
npm init
npm install --save botkit
```

## BotUserの作成

1. 左のチーム名をクリックすると出てくるメニューから「Apps & Custom Integrations」を選択する。
 ![](slack-iw-001.png)

2. Botsを探してクリックする。
 ![](slack-bu-002.png)

3. Installボタンをクリックする。
 ![](slack-bu-003.png)

4. Botの名前を入力する。
 ![](slack-bu-004.png)

5. API Tokenを覚えておく。
 ![](slack-bu-005.png)
