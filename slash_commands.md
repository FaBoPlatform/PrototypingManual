# Slash Commands

/から始まるコマンドを自作できる。  
コマンドが入力されると指定のアドレスにPOSTされる。


## Slackの設定

1. Integrationを追加する。  
  左のチーム名をクリックすると出てくるメニューから「Apps & Custom Integrations」を選択する。
 ![](slack-iw-001.png)

2. Slash Commandsを探してクリックする。
 ![](slack-sl-002.png)

3. Installボタンをクリックする。
 ![](slack-sl-003.png)

4. コマンド名を決める。
 ![](slack-sl-004.png)

5. コマンド実行の通知先URLを設定する。
 ![](slack-sl-005.png)
